# link shortener

Shorts or links. This is an old proof of concept

## installation

* Install python virtualenv
  ```bash
  python3 -m venv venv
  ```
  
* Source virtualenv
  ```bash
  source venv/bin/activate
  ```
  
* Install requirements
  ```bash
  pip install --require-hashes -r requirements.txt
  ```

## usage

* Start the server
  ```bash
  python3 __main__.py
  ```

* Goto localhost:5000 to see the result
